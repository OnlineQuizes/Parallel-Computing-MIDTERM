Parallel Computing Quiz

1. A serial program executes:

Correct Answer: b) One instruction at a time

2. A parallel program executes:

Correct Answer: b) Multiple instructions at the same time

3. All processors are made of:

Correct Answer: b) Transistors

4. Smaller transistors mean:

Correct Answer: b) More transistors and higher speed

5. Why did processors begin using more cores in modern CPUs?

Correct Answer: d) Because sequential performance stopped improving

6. Faster processors use more power, create more heat, and can become unreliable.

Correct Answer: a) True

7. Why did processors require heatsinks in modern computers?

Correct Answer: a) Because they generated a large amount of heat

8. Why do we need ever-increasing computer performance?

Correct Answer: a) To solve complex problems in science, medicine, and data analysis

9. Parallel programming makes programs faster by using many cores together.

Correct Answer: a) True

10. Data parallelism means:

Correct Answer: c) Same work on different parts of data

11. Task parallelism means:

Correct Answer: b) Different tasks at the same time

12. Which is example of data parallelism?

Correct Answer: c) Divide the exams among workers

13. Which is example of task parallelism?

Correct Answer: a) Divide the questions among workers

14. Why do parallel programs need coordination between cores?

Correct Answer: d) All of the above

15. Explicit parallelism means:

Correct Answer: a) Programmer writes parallel code manually

16. Implicit parallelism means:

Correct Answer: d) It is automatic and easier to write

17. In parallel processing, communication means:

Correct Answer: b) Sending partial results between cores

18. In parallel processing, load balancing means:

Correct Answer: c) Sharing the work evenly among the cores

19. In parallel processing, synchronization means:

Correct Answer: a) Making sure cores stay in step and don't get too far ahead

20. In a shared-memory system:

Correct Answer: a) All cores share the same memory

21. In a distributed-memory system:

Correct Answer: b) Each core has its own private memory

22. Concurrent computing allows multiple tasks to be in progress at the same time.

Correct Answer: a) True

23. Parallel computing runs one task at a time to solve simple problem.

Correct Answer: b) False

Explanation: Parallel computing runs multiple tasks at the same moment to solve one problem together.

24. In distributed computing, a program may need to cooperate with other programs to solve a problem.

Correct Answer: a) True

25. The Von Neumann architecture consists mainly of:

Correct Answer: c) CPU, Memory, and Interconnect (Bus)

26. The Von Neumann bottleneck refers to:

Correct Answer: b) The connection limits between CPU and memory

27. The CPU (Central Processing Unit) is divided into:

Correct Answer: c) Control Unit and Datapath

28. The Control Unit in the CPU is responsible for:

Correct Answer: a) Controls the sequence of instruction execution

29. The Datapath in the CPU:

Correct Answer: b) Performs arithmetic and logical operations

30. The Bus in the Von Neumann architecture:

Correct Answer: a) Transfers data and instructions between CPU and memory

31. Registers are large and slow memory units used for long-term data storage.

Correct Answer: b) False

Explanation: Registers are very fast, small storage units inside the CPU used for temporary data.

32. What does the Program Counter (PC) do?

Correct Answer: c) Holds the address of the next instruction

33. When data are transferred from memory to CPU, the process is called:

Correct Answer: a) Fetch or Read

34. When data are transferred from CPU to memory, it is called:

Correct Answer: b) Write or Store

35. Registers are slower than main memory.

Correct Answer: b) False

Explanation: Registers are faster than main memory.

36. The main purpose of an Operating System (OS) is to:

Correct Answer: b) Manage hardware and software resources

37. What is a process in computing?

Correct Answer: d) An instance of a program currently being executed

38. Which of the following is NOT a component of a process?

Correct Answer: c) Printer driver

39. When a process is ready, it means:

Correct Answer: a) It's prepared to run but waiting for CPU time

40. When a process is waiting, it means:

Correct Answer: c) It's paused until an event occurs (like I/O input)

41. What is Multitasking?

Correct Answer: a) Running multiple processes at the same time

42. Which of the following is true about threads?

Correct Answer: b) The smallest unit of execution within a process

43. Most modern operating systems are single-tasking.

Correct Answer: b) False

Explanation: Most modern operating systems are multitasking.

44. What is the main purpose of hardware multithreading?

Correct Answer: a) To keep the CPU busy when a thread is stalled

45. Processes are lighter and faster than threads.

Correct Answer: b) False

Explanation: Threads are lighter and faster than processes.

46. A process and a program are exactly the same thing.

Correct Answer: b) False

Explanation: A process is a running instance of a program.

47. Starting a new thread is called forking and ending it is called joining.

Correct Answer: a) True

48. A CPU cache is best described as:

Correct Answer: c) Small and fast memory that stores used data

49. Virtual memory primarily acts as:

Correct Answer: c) A cache for secondary storage (disk/SSD)

50. Which parallelism executes multiple instructions in the same thread?

Correct Answer: b) Instruction-Level Parallelism (ILP)

51. Which parallelism executes multiple threads at the same time?

Correct Answer: a) Thread-Level Parallelism (TLP)

52. A processor that supports dynamic multiple issue is often called:

Correct Answer: b) Superscalar

53. Which type of multithreading switches threads after each instruction?

Correct Answer: a) Fine-Grained Multithreading

54. The main goal of both software and hardware techniques is to exploit:

Correct Answer: d) Instruction-Level Parallelism (ILP)

55. Which type of multithreading switches threads only during long stalls?

Correct Answer: b) Coarse-Grained Multithreading

56. Which type of multithreading allows multiple threads to execute instructions at the same time?

Correct Answer: c) Simultaneous Multithreading (SMT)

57. Which parallel hardware is usually not visible to the programmer?

Correct Answer: a) Multiple issue and pipelining

58. A basic parallel computer architecture mainly consists of:

Correct Answer: d) All of above

59. Modern GPUs often optimize performance using:

Correct Answer: b) SIMD

60. In MIMD systems, processors:

Correct Answer: b) Execute different instructions on different data

61. SIMD is best suited to support:

Correct Answer: b) Data parallelism

62. A vector processor primarily operates on:

Correct Answer: c) Data vectors

63. A major advantage of vector processors is that they are generally:

Correct Answer: b) Improved memory utilization

64. Which of the following best describes MIMD systems?

Correct Answer: d) The most common modern architecture

65. Which architecture is best suited for tasks such as image processing and matrix operations?

Correct Answer: b) SIMD

66. One major drawback of SIMD architecture is:

Correct Answer: c) Some processing units remain idle

67. Which term describes the time it takes for data to travel between processor and memory?

Correct Answer: c) Latency

68. Which term refers to the amount of data that can be transferred between processor and memory in a given time?

Correct Answer: b) Bandwidth

69. A major challenge of shared memory systems is:

Correct Answer: b) Contention among processors

70. In a UMA system:

Correct Answer: c) All processors access memory at the same speed

71. An advantage of UMA is:

Correct Answer: b) Low latency

72. A disadvantage of UMA systems is:

Correct Answer: b) Limited Scalability

73. NUMA are also called SMP (symmetric multiprocessor) because memory access is balanced.

Correct Answer: b) False

Explaination: UMA systems are also called SMP systems because memory access is balanced.

74. What is true about NUMA systems?

Correct Answer: b) Processors access their local memory faster than remote memory

75. A main advantage of NUMA is:

Correct Answer: b) Reduced contention on memory

76. Which statement best describes speculation in processors?

Correct Answer: b) Executing instructions in advance based on assumptions

77. A main disadvantage of NUMA is:

Correct Answer: d) All of above

78. A common example of hybrid distributed-shared memory systems is:

Correct Answer: b) Cluster of computers

79. Which part affects the performance of both distributed and shared memory systems?

Correct Answer: c) Interconnection networks

80. What is a bus interconnect in shared memory systems?

Correct Answer: b) A collection of shared communication wires controlled by hardware

81. What is an advantage of bus interconnects?

Correct Answer: c) Low cost and flexibility

82. What is a switched interconnect?

Correct Answer: b) Uses switches to route data between connected devices

83. What is true about switched interconnects compared to buses?

Correct Answer: b) They provide faster communication

84. Which of the following are examples of indirect interconnection networks?

Correct Answer: b) Crossbar and Omega network

85. Which of the following are examples of direct interconnection networks?

Correct Answer: b) Ring and Toroidal Mesh

86. The formula to calculate message transfer time is:

Correct Answer: b) Time = Latency + (Message Size / Bandwidth)

87. Which network is a highly connected direct interconnection used in actual systems?

Correct Answer: d) Hypercube network

88. Which network type connects devices in a loop?

Correct Answer: c) Ring network

89. Which network type allows multiple paths between devices, increasing complexity?

Correct Answer: c) Toroidal Mesh

90. Temporal locality refers to:

Correct Answer: b) Accessing the same data again in the near future

91. Spatial locality refers to:

Correct Answer: a) Accessing data near the recently accessed data

92. Which cache level is smallest and fastest?

Correct Answer: c) L1

93. A cache miss occurs when:

Correct Answer: c) The requested data is not found in the cache

94. Which write strategy updates the main memory at the same time the data is written to the cache?

Correct Answer: b) Write-through

95. Which cache write strategy marks data as dirty and writes it to main memory only when the cache line is replaced?

Correct Answer: c) Write-back

96. Cache coherence ensures that:

Correct Answer: a) All cores see the same data in their caches

97. False sharing occurs when:

Correct Answer: b) Two cores access different data on the same cache line

98. One way to reduce false sharing is:

Correct Answer: c) Apply padding to separate data into different cache lines

99. Which technique is better for large multicore systems to maintain cache coherence?

Correct Answer: b) Directory-based coherence

100. What issue can occur in caches?

struct data {
    int x;
    int y;
};
struct data shared;
void core1() { shared.x = 10; }
void core2() { shared.y = 20; }
Correct Answer: b) False sharing because a and b are in the same cache line

101. Who manages CPU caches in a system?

Correct Answer: b) System hardware

102. Removing false sharing will:

Correct Answer: b) Improve performance
